# Elite Universal V10.6.7 🌍

Un script universal para Roblox optimizado para dispositivos móviles y PC, diseñado con una interfaz moderna inspirada en juegos competitivos.

## 🚀 Características
- **Main:** CFrame Speed, Salto Infinito, SpinBot y Phase (Atravesar muros).
- **Visuals:** Infinite Zoom, FOV ajustable y ESP (Wallhack).
- **Misc:** Reproductor de emotes de la comunidad y Server Hop.
- **Settings:** Anti-AFK, Auto-Clicker y Modo Papa (FPS Boost).
